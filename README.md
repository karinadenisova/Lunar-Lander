Lunar Lander – Консольная версия на 7 языках
Классическая аркада «Лунный ровер» – посадите модуль на поверхность Луны с ограниченным топливом!
Управляйте тягой, следите за скоростью и высотой, чтобы совершить мягкую посадку.
Реализована на семи языках программирования с единым геймплеем, цветной графикой, системой очков и рекордов, а также режимом автоматической посадки.

🚀 Возможности
Реалистичная физика – гравитация, тяга двигателя, инерция.

Управление – стрелки влево/вправо для наклона, пробел для включения двигателя, Q для выхода.

Цель – совершить мягкую посадку на посадочную площадку (скорость < 3 м/с, угол < 10°).

Ограниченное топливо – расходуется при работе двигателя; при нуле – только свободное падение.

Счёт – бонус за оставшееся топливо и точность посадки.

Рекорд – лучший результат сохраняется в файл.

Режимы сложности – гравитация Луны, Земли, Марса (выбор в начале игры).

Автопилот – опция автоматической посадки (демонстрация).

Цветная графика – лунная поверхность, посадочный модуль, индикаторы.

Звуковые эффекты – при касании поверхности (ASCII bell).

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [опции]
Опции
Опция	Описание
-g, --gravity <planet>	Выбор гравитации: moon, earth, mars (по умолчанию moon)
-a, --autopilot	Включить режим автопилота (компьютер управляет посадкой)
-h, --help	Справка
Управление
Клавиша	Действие
← / A	Наклон влево
→ / D	Наклон вправо
Пробел	Включить / выключить двигатель
R	Перезапуск после посадки или крушения
Q	Выход из игры
🛠 Установка и запуск
Python
bash
pip install windows-curses  # только для Windows
python lunar.py [опции]
C++
bash
g++ -std=c++17 lunar.cpp -o lunar -lncurses
./lunar [опции]
Требуется библиотека ncurses (установите через пакетный менеджер).

Go
bash
go get github.com/nsf/termbox-go
go build lunar.go
./lunar [опции]
JavaScript (Node.js)
bash
npm install blessed
node lunar.js [опции]
C#
bash
csc lunar.cs
mono lunar.exe [опции]   # или dotnet run
Требуется поддержка ANSI в терминале (на Windows используйте Console.OutputEncoding = Encoding.UTF8).

Ruby
bash
ruby lunar.rb [опции]
Java
bash
javac -cp .:lanterna.jar lunar.java
java -cp .:lanterna.jar lunar [опции]
Требуется библиотека lanterna (скачайте JAR и добавьте в classpath).

🧠 Логика игры
Модуль – находится в верхней части экрана, имеет положение (x, y), скорость (vx, vy), угол наклона.

Управление – наклон меняет направление тяги, двигатель создаёт ускорение в направлении наклона.

Гравитация – постоянное ускорение вниз (зависит от выбранной планеты).

Топливо – уменьшается при работе двигателя; при нуле двигатель отключается.

Посадка – при касании поверхности проверяется вертикальная и горизонтальная скорость, а также угол.

Условия – мягкая посадка (v < 3 м/с, угол < 10°), жёсткая (v >= 3 м/с) – крушение.

Счёт – 1000 очков за мягкую посадку + бонус за оставшееся топливо (×10).

Рекорд – сохраняется в файл ~/.lunar_record.json.

✨ Дополнительные фичи
Автопилот – автоматическое управление для демонстрации (реализовано простым ПИД-регулятором).

Индикаторы – отображение высоты, скорости, топлива, угла, состояния двигателя.

Анимация – плавное движение модуля с перерисовкой.

Цветовая схема – красный для опасной скорости, зелёный для безопасной.

Звук – при касании поверхности.
